Sobre o Meu App

Descrição do Projeto

Este aplicativo é uma plataforma para gerenciar usuários e suas tarefas. Os usuários podem ser cadastrados, editados e deletados, e cada usuário pode ter várias tarefas associadas.

Funcionalidades

  • Cadastro de usuários
  • Edição de informações de usuários
  • Remoção de usuários
  • Cadastro de tarefas vinculadas a usuários
  • Visualização de usuários e suas tarefas

Como Contribuir

Se você deseja contribuir para este projeto, sinta-se à vontade para abrir uma issue ou enviar um pull request.
